# N5 Vocabulary Canonical Forms - Decision Table

## Decision Principles
1. **Modern usage first** - How native speakers write today
2. **N5-appropriate** - Match actual JLPT N5 expectations  
3. **Learner clarity** - Reduce confusion, teach practical forms
4. **Consistent rules** - Clear guidelines students can follow
