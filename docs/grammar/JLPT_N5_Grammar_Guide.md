# Part 1: Complete Particle Guide

## 1.1 は (wa) — Topic Marker

**📌 RULE:** Marks what the sentence is ABOUT. Pronounced 'wa' (not 'ha').

### Patterns:
- **[Noun] は [Comment]** → 'As for [Noun]...'
- **[Noun] は [Noun] です** → X is Y
- **[Noun] は [Adj] です** → X is [adjective]

### Examples:
- 私は 学生です。 → As for me, (I) am a student.
- 東京は 大きいです。 → As for Tokyo, (it) is big.
- この本は おもしろいです。 → This book is interesting.

### Special Use - Contrast:
- 肉は 食べますが、魚は 食べません。 → I eat meat, but I don't eat fish.

---

## 1.2 が (ga) — Subject Marker

**📌 RULE:** Marks WHO/WHAT does something. Used for new info, emphasis, abilities, likes.

### が is REQUIRED with:

**① Question words:**
- だれが 来ますか。 → WHO is coming?
- 何が ありますか。 → WHAT is there?

**② Existence (います/あります):**
- 猫が います。 → There IS a cat.
- 本が あります。 → There IS a book.

**③ Likes/Wants/Abilities:**
- 日本語が わかります。 → I understand Japanese.
- 寿司が 好きです。 → I like sushi.
- 水が 欲しいです。 → I want water.
- ピアノが 上手です。 → I'm good at piano.

### は vs が Comparison:

| は (Topic) | が (Subject) |
|------------|--------------|
| Old/known information | New/important information |
| 'As for X...' | 'X is the one that...' |
| 田中さんは 学生です。 | 田中さんが 学生です。 |
| (Talking about Tanaka) | (Tanaka is THE student) |

---

## 1.3 を (o) — Object Marker

**📌 RULE:** Marks the DIRECT OBJECT — the thing receiving the action.

### Pattern:
- **[Object] を [Verb]** → do [verb] to [object]

### Examples:
- 水を 飲みます。 → I drink water.
- 本を 読みます。 → I read a book.
- 日本語を 勉強します。 → I study Japanese.
- テレビを 見ます。 → I watch TV.

### Movement Through:
- 公園を 歩きます。 → I walk through the park.
- 橋を 渡ります。 → I cross the bridge.

### Point of Departure:
- 家を 出ます。 → I leave the house.
- 電車を 降ります。 → I get off the train.

---

## 1.4 に (ni) — Multi-Purpose Particle

**📌 RULE:** 7+ uses: direction, time, location, recipient, purpose, frequency, result.

### Use 1: Direction/Destination
**[Place] に 行きます/来ます** → go/come TO [place]
- 日本に 行きます。 → I go TO Japan.
- 家に 帰ります。 → I return home.

### Use 2: Location of Existence
**[Place] に います/あります** → exist AT [place]
- 机の上に 本が あります。 → There is a book ON the desk.
- 部屋に 猫が います。 → There is a cat IN the room.

### Use 3: Specific Time
**[Time] に [Verb]** → do [verb] AT [time]
- 7時に 起きます。 → I wake up AT 7.
- 月曜日に 会います。 → I'll meet ON Monday.
- 3月に 卒業します。 → I graduate IN March.

**⚠️ DON'T use に with:** 今日、昨日、明日、毎日、いつ (relative time)

### Use 4: Recipient
**[Person] に [Object] を あげます** → give [object] TO [person]
- 友達に プレゼントを あげます。 → I give a present TO my friend.

### Use 5: Purpose
**[Verb stem] に 行きます** → go TO [do verb]
- 買い物に 行きます。 → I go shopping.
- 映画を 見に 行きます。 → I go to watch a movie.

### Use 6: Frequency
**[Time] に [Number] 回** → [number] times PER [time]
- 週に 3回 運動します。 → I exercise 3 times a week.

### Use 7: Result/Becoming
**[Noun] に なります** → become [noun]
- 医者に なります。 → I'll become a doctor.

---

## 1.5 で (de) — Action Context

**📌 RULE:** WHERE action happens, HOW it's done, or WHY.

### Use 1: Place of Action
**[Place] で [Action]** → do [action] AT [place]
- 図書館で 勉強します。 → I study AT the library.
- レストランで 食べます。 → I eat AT the restaurant.

### で vs に Comparison:

| に (Existence) | で (Action) |
|----------------|-------------|
| 猫が 部屋に います。 | 部屋で 本を 読みます。 |
| (Cat exists in room) | (Read books in room) |

### Use 2: Means/Method/Tool
**[Tool] で [Verb]** → do [verb] BY/WITH [tool]
- 箸で 食べます。 → I eat WITH chopsticks.
- 日本語で 話します。 → I speak IN Japanese.
- 電車で 行きます。 → I go BY train.

### Use 3: Reason/Cause
- 病気で 休みます。 → I'm absent DUE TO illness.
- 雨で 中止です。 → Cancelled DUE TO rain.

### Use 4: Scope
**[Scope] で 一番** → most/best IN [scope]
- 日本で 一番 高い山 → The highest mountain IN Japan

### Use 5: Total
- 全部で 1000円です。 → 1000 yen IN TOTAL.
- 3人で 行きます。 → We'll go as 3 people.

---

## 1.6 の (no) — Connection Particle

**📌 RULE:** Connects nouns: possession, description, categories.

### Use 1: Possession
**[Owner] の [Thing]** → [owner]'s [thing]
- 私の 本 → my book
- 田中さんの 車 → Tanaka's car
- 日本の 文化 → Japanese culture

### Use 2: Category/Type
**[Category] の [Noun]** → [noun] of [category]
- 日本語の 先生 → Japanese language teacher
- 経済の 本 → economics book

### Use 3: Location
**[Place] の [Noun]** → [noun] at [place]
- 東京の ホテル → hotel in Tokyo
- 机の 上 → on top of the desk
- 駅の 前 → in front of the station

### Use 4: Apposition (A = B)
- 友達の 田中さん → my friend Tanaka-san
- 医者の 山田さん → Dr. Yamada

### Use 5: Nominalization
**[Verb/Adj] の** → the act of / the one that
- 食べるのが 好きです。 → I like eating.
- 赤いのを ください。 → Please give me the red one.
- 高いのは 買いません。 → I won't buy expensive ones.

### Chaining:
- 私の 友達の 車 → my friend's car
- 日本の 大学の 先生 → teacher at a Japanese university

---

## 1.7 と (to) — And/With

### Use 1: And (complete list)
**[A] と [B]** → [A] AND [B] (only these)
- りんごと みかんを 買いました。 → I bought apples AND oranges (only these).

### Use 2: With (accompaniment)
**[Person] と [Verb]** → do [verb] WITH [person]
- 友達と 映画を 見ます。 → I watch movies WITH friends.
- だれと 行きますか。 → WHO are you going WITH?

### Use 3: Comparison
**[A] と [B] と どちらが** → Which is more, A or B?
- 犬と 猫と どちらが 好きですか。 → Which do you like more?

### Use 4: Quotation
**「...」と 言います** → say '...'
- 「ありがとう」と 言いました。 → I said 'thank you.'

---

## 1.8 や (ya) — Partial List

**📌 RULE:** Lists examples (non-exhaustive). Implies 'etc.'

**[A] や [B] (など)** → [A], [B], etc.
- 本や 雑誌を 読みます。 → I read books, magazines, etc.

### と vs や:

| と (Complete) | や (Partial) |
|---------------|--------------|
| りんごと みかん | りんごや みかん |
| (only apples and oranges) | (apples, oranges, and others) |

---

## 1.9 から/まで — From/Until

### から — From (Starting Point)
**[Time/Place] から** → FROM [time/place]
- 9時から 働きます。 → I work FROM 9.
- 東京から 来ました。 → I came FROM Tokyo.

### まで — Until/To (Ending Point)
**[Time/Place] まで** → UNTIL/TO [time/place]
- 5時まで 働きます。 → I work UNTIL 5.
- 駅まで 歩きます。 → I walk TO the station.

### Combined:
**[Start] から [End] まで** → FROM [start] TO [end]
- 9時から 5時まで 働きます。 → I work FROM 9 TO 5.

### から — Because
**[Reason] から** → because [reason]
- 暑いから、窓を 開けます。 → BECAUSE it's hot, I'll open the window.

---

## 1.10 も (mo) — Also/Too

**📌 RULE:** Replaces は/が/を to mean 'also' or 'too'.

**[Noun] も** → [noun] ALSO/TOO
- 私も 学生です。 → I am ALSO a student.
- これも おいしいです。 → This is ALSO delicious.
- 日本語も 英語も 話せます。 → I can speak BOTH Japanese AND English.

### With Question Words + Negative:
- だれも いません。 → NO ONE is here.
- 何も 食べません。 → I don't eat ANYTHING.
- どこも 行きません。 → I don't go ANYWHERE.

---

## 1.11 か (ka) — Question/Or

### Use 1: Question Marker
- 学生ですか。 → Are you a student?
- これは 何ですか。 → What is this?

### Use 2: Or
**[A] か [B]** → [A] OR [B]
- コーヒーか 紅茶 → coffee OR tea

### Use 3: Uncertainty (some-)
- だれか 来ました。 → SOMEONE came.
- 何か 食べましょう。 → Let's eat SOMETHING.
- どこかに 行きたいです。 → I want to go SOMEWHERE.

---

## 1.12 ね/よ — Sentence Endings

### ね — Seeking Agreement
**[Statement] ね** → ..., right? / ..., isn't it?
- いい天気ですね。 → Nice weather, ISN'T IT?
- そうですね。 → That's right.

### よ — Assertion
**[Statement] よ** → I'm telling you... / you know!
- おいしいですよ！ → It's delicious, YOU KNOW!
- 危ないですよ！ → It's dangerous! (warning)

💡 **TIP:** ね = seeking shared understanding. よ = telling new info.

# Part 2: Pronouns & Demonstratives

## 2.1 Personal Pronouns

| | Casual | Polite | Notes |
|---|--------|--------|-------|
| I (male) | 僕 (boku) | 私 (watashi) | 俺 = very casual |
| I (female) | 私 (watashi) | 私 (watashi) | あたし = casual |
| You | 君 (kimi) | あなた | Use name+さん! |
| He | 彼 (kare) | あの人 | 彼 = boyfriend too |
| She | 彼女 (kanojo) | あの人 | 彼女 = girlfriend too |
| We | 僕たち | 私たち | Add たち for plural |

**⚠️ CAUTION:** Avoid あなた! Use the person's name + さん instead.

## 2.3 Question Words

| Word | Meaning | Example |
|------|---------|---------|
| 何 (なに/なん) | What | これは 何ですか。 |

# Part 3: Verb Conjugations

## 3.1 Polite Forms (ます form)

| Form | 食べる (eat) | 行く (go) | する (do) | 来る (come) |
|------|-------------|----------|-----------|------------|
| Present | 食べます | 行きます | します | 来ます |
| Past | 食べました | 行きました | しました | 来ました |
| Negative | 食べません | 行きません | しません | 来ません |
| Neg Past | 食べませんでした | 行きませんでした | しませんでした | 来ませんでした |

---

## 3.2 Te-Form (て形)

**📌 RULE:** Essential for requests, progressive, connecting actions.

| Ending | Change | Example |
|--------|--------|---------|
| う・つ・る | → って | 買う→買って, 待つ→待って, 帰る→帰って |
| む・ぶ・ぬ | → んで | 飲む→飲んで, 遊ぶ→遊んで |
| く | → いて | 書く→書いて, 聞く→聞いて |
| ぐ | → いで | 泳ぐ→泳いで |
| す | → して | 話す→話して |
| る (Group 2) | → て | 食べる→食べて, 見る→見て |
| Irregular | | する→して, 来る→来て, 行く→行って |

### Te-Form Uses:

**① Request: ～てください**
- ちょっと 待ってください。 → Please wait a moment.
- ゆっくり 話してください。 → Please speak slowly.

**② Progressive: ～ています**
- 今 食べています。 → I am eating now.
- 東京に 住んでいます。 → I live in Tokyo.

**③ Connecting: ～て、～**
- 朝 起きて、シャワーを あびます。 → I wake up and take a shower.

**④ Permission: ～てもいいですか**
- 写真を 撮ってもいいですか。 → May I take a photo?

---

## 3.3 Want to Do: ～たいです

**[Verb stem] + たいです** → want to [verb]
- 日本に 行きたいです。 → I want to go to Japan.
- 何を 食べたいですか。 → What do you want to eat?

**Negative:**
- 行きたくないです。 → I don't want to go.

# Part 4: Adjective Rules

## 4.1 Two Types

| | い-Adjectives | な-Adjectives |
|---|---------------|---------------|
| ID | End in い: 高い、大きい | Don't end in い: 静か、好き |
| Before Noun | 高い 山 | 静かな 部屋 |
| Sentence End | 高いです | 静かです |
| Negative | 高くないです | 静かじゃないです |
| Past | 高かったです | 静かでした |
| Neg Past | 高くなかったです | 静かじゃなかったです |

**⚠️ EXCEPTION:** きれい, 嫌い, 有名 END in い but are な-adjectives!

**⚠️ IRREGULAR:** いい (good) → よくない → よかった → よくなかった

---

## 4.2 Common Adjectives

### い-Adjectives:
| | | | |
|---|---|---|---|
| 大きい (big) | 小さい (small) | 高い (tall/expensive) | 安い (cheap) |
| 新しい (new) | 古い (old) | いい (good) | 悪い (bad) |
| 暑い (hot-weather) | 寒い (cold-weather) | おいしい (delicious) | 楽しい (fun) |
| 難しい (difficult) | 易しい (easy) | 忙しい (busy) | 近い (near) |

### な-Adjectives:
| | | | |
|---|---|---|---|
| 静か (quiet) | きれい (beautiful) | 元気 (healthy) | 好き (liked) |
| 嫌い (disliked) | 上手 (skillful) | 下手 (unskillful) | 便利 (convenient) |
| 有名 (famous) | 簡単 (simple) | 大切 (important) | 親切 (kind) |

# Quick Reference: Particle Summary

| P | Main Use | Pattern | Example |
|---|----------|---------|---------|
| は | Topic | N は [comment] | 私は 学生です。 |
| が | Subject/New info | N が V | 猫が います。 |
| を | Direct object | N を V | 本を 読みます。 |
| に | Direction/Time/Location | N に V | 7時に 起きます。 |
| で | Place of action/Means | N で V | 電車で 行きます。 |
| の | Possession | N の N | 私の 本 |
| と | And/With | N と N / N と V | 友達と 行きます。 |
| や | Partial list | N や N | 本や 雑誌 |
| から | From/Because | N から | 9時から |
| まで | Until/To | N まで | 5時まで |
| も | Also/Too | N も | 私も 学生です。 |
| か | Question/Or | S か | 学生ですか。 |
| ね | Right? | S ね | いい天気ですね。 |
| よ | Assertion | S よ | おいしいですよ！ |
