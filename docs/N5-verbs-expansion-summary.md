## Known Special Cases

### 1. Homonyms Handled in Main Dataset

These verbs are **separate from homonyms already in N5 dataset**:
- 違う (be different) - NOT 帰る (return home)
- 戻る (return/go back) - distinct from 帰る (return home)

### 2. Similar Verbs Clarified

| This Dataset | Main N5 Dataset | Difference |
|--------------|-----------------|------------|
| 茹でる (boil food) | - | Transitive - you boil something |
| 沸く (water boils) | - | Intransitive - water boils itself |
| 仕事する (work) | 働く (work) | 仕事する is compound, 働く is simple |
| 料理する (cook) | 作る (make) | 料理する is specific to food |

### 3. Suru-Verbs

9 irregular suru-verbs included:
- 仕事する, ドライブする, 発表する, 化粧する, けんかする, 休憩する, 旅行する, 料理する

All follow irregular conjugation pattern (suru → shimasu → shite).
