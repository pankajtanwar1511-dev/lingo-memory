# Japanese Office Practice — Rapid Fluency Guide

**Purpose: Active production training — phrases you write and say, not just recognize**

> Companion to `japanese-office-vocabulary.md`. Where the reference file lists vocabulary, this file gives you drillable sentence patterns, situation templates, and a structured 30-day plan.

---

## How to Use This File

1. **Phase 1 Survival Core** — Learn these 120 words first. Everything else builds on them.
2. **Communication Pattern Drills** — Practice short business messages by pattern type.
3. **Situation Templates** — Standup / Incident / 1-on-1 / HR ready-to-use sentence frames.
4. **30-Day Plan** — One focused topic per day with clear weekly goals.

> **Production rule:** If you can't write the sentence in 5 seconds, keep drilling. Passive recognition is not enough.

## Section 2 — Communication Pattern Drills

Practice by pattern type. Each pattern has 3–5 variations. Drill until automatic.

> **How to drill:** Cover the Japanese column. Read the English. Write the Japanese from memory. Repeat.

### 3A — Daily Standup / Progress Report

**Context: Daily team sync. Short updates — 30 seconds to 2 minutes per person.**

---

**Opening frame:**

```
〜について報告します。
(~ についてほうこくします。)
I'll report on ~.
```

```
本日の進捗を共有します。
(ほんじつのしんちょくをきょうゆうします。)
I'll share today's progress.
```

---

**Progress — on track:**

```
〜は完了しました。
(~ はかんりょうしました。)
~ is complete.
```

```
〜は予定通り進んでいます。
(~ はよていどおりすすんでいます。)
~ is progressing as scheduled.
```

```
本日は〜を予定しています。
(ほんじつは~ をよていしています。)
Today I plan to work on ~.
```

---

**Progress — delayed:**

```
〜は少し遅れています。
(~ はすこしおくれています。)
~ is running a bit behind.
```

```
〜で遅延が発生しています。
(~ でちえんがはっせいしています。)
A delay has occurred with ~.
```

```
〜について手間取っています。
(~ についててまどっています。)
I'm taking longer than expected on ~.
```

```
本日中に修正予定です。
(ほんじつちゅうにしゅうせいよていです。)
Plan to fix / complete it by end of day.
```

---

**Blockers / asking for help:**

```
〜について相談があります。
(~ についてそうだんがあります。)
I have something to discuss about ~.
```

```
〜で詰まっています。
(~ でつまっています。)
I'm stuck on ~.
```

```
〜について確認が必要です。
(~ についてかくにんがひつようです。)
I need to confirm ~.
```

```
〜についてアドバイスをいただけますか？
(~ についてアドバイスをいただけますか？)
Could I get your advice on ~?
```

---

**Standup closing frame:**

```
以上です。よろしくお願いします。
(いじょうです。よろしくおねがいします。)
That's all. Thank you.
```

---

**Full example standup:**

```
本日の進捗を共有します。
〇〇機能の実装は完了しました。
現在、〇〇のテストを対応中です。
〇〇について少し手間取っていますが、本日中に完了予定です。
以上です。よろしくお願いします。
```

### 3C — Incident Response

**Context: System failure, defect, or urgent operational problem. Speed and clarity matter.**

---

**Phase 1 — Detection / Initial Alert**

```
【障害発生】〜においてシステム障害が発生しました。
(【しょうがいはっせい】〜においてシステムしょうがいがはっせいしました。)
[INCIDENT] A system failure has occurred in ~.
```

```
〜系でエラーが検知されました。現在、影響範囲を確認中です。
(~ けいでエラーがけんちされました。げんざい、えいきょうはんいをかくにんちゅうです。)
An error has been detected in the ~ system. Currently confirming scope of impact.
```

```
〜が正常に動作していません。原因を調査しています。
(~ がせいじょうにどうさしていません。げんいんをちょうさしています。)
~ is not functioning normally. Investigating the cause.
```

---

**Phase 2 — Status Updates (send every 30–60 min)**

```
現在、〜の原因を調査しています。続報をお待ちください。
(げんざい、〜のげんいんをちょうさしています。ぞくほうをおまちください。)
Currently investigating the cause of ~. Please wait for further updates.
```

```
影響範囲：〜
対応状況：調査中
(えいきょうはんい：〜／たいおうじょうきょう：ちょうさちゅう)
Scope of impact: ~ / Status: Under investigation
```

```
暫定対応として、〜を実施しました。引き続き対応中です。
(ざんていたいおうとして、〜をじっしました。ひきつづきたいおうちゅうです。)
As a temporary fix, we have implemented ~. Handling continues.
```

```
〜の切り戻しを実施しました。
(〜のきりもどしをじっしました。)
We have executed a rollback of ~.
```

---

**Phase 3 — Resolution**

```
〜が復旧しました。ご迷惑をおかけして申し訳ございませんでした。
(〜がふっきゅうしました。ごめいわくをおかけしてもうしわけございませんでした。)
~ has been restored. We sincerely apologize for the inconvenience.
```

```
復旧完了しました。現在、正常稼働を確認しています。
(ふっきゅうかんりょうしました。げんざい、せいじょうかどうをかくにんしています。)
Restoration is complete. Currently confirming normal operation.
```

```
詳細は後ほど障害報告書をご共有します。
(しょうさいはのちほどしょうがいほうこくしょをごきょうゆうします。)
We will share the full incident report later.
```

```
再発防止策については、別途ご報告します。
(さいはつぼうしさくについては、べっとごほうこくします。)
We will report separately on measures to prevent recurrence.
```

---

**Escalation:**

```
対応が難しい状況のため、エスカレーションさせていただきます。
(たいおうがむずかしいじょうきょうのため、エスカレーションさせていただきます。)
Escalating as the situation is difficult to handle alone.
```

```
〜について、上長への報告が必要と判断しました。
(〜について、じょうちょうへのほうこくがひつようとはんだんしました。)
I have determined that we need to report to management about ~.
```

---

**Severity classification phrases:**

| Phrase | Kana | Meaning |
|---|---|---|
| 重大な障害です | じゅうだいなしょうがいです | This is a critical incident |
| 軽微な問題です | けいびなもんだいです | This is a minor issue |
| 影響なし | えいきょうなし | No impact |
| サービス全体に影響があります | サービスぜんたいにえいきょうがあります | Affecting the entire service |
| 一部ユーザーに影響があります | いちぶユーザーにえいきょうがあります | Affecting some users |

### 3D — 1-on-1 Meetings

**Context: Regular check-in with your direct manager. Structured as report → discuss → align.**

---

**Opening:**

```
よろしくお願いします。最近の進捗を報告させてください。
(よろしくおねがいします。さいきんのしんちょくをほうこくさせてください。)
Thank you. Let me report on my recent progress.
```

---

**Progress report:**

```
現在、〜を担当しています。
(げんざい、〜をたんとうしています。)
I'm currently in charge of ~.
```

```
〜は予定通り進んでいます。
(〜はよていどおりすすんでいます。)
~ is progressing as scheduled.
```

```
〜は少し遅れていますが、〇〇日までには完了できます。
(〜はすこしおくれていますが、〇〇にちまでにはかんりょうできます。)
~ is a bit behind, but I can complete it by ~.
```

---

**Blockers / concerns:**

```
〜について相談があります。
(〜についてそうだんがあります。)
I have something to discuss about ~.
```

```
〜で詰まっていまして、ご意見をいただけますか？
(〜でつまっていまして、ごいけんをいただけますか？)
I'm stuck on ~ and would appreciate your input.
```

```
〜についてどう進めればよいか確認させてください。
(〜についてどうすすめればよいかかくにんさせてください。)
Let me confirm how to proceed with ~.
```

---

**Career / goals:**

```
〜について学びたいと思っています。
(〜についてまなびたいとおもっています。)
I'd like to learn about ~.
```

```
今後は〜に取り組みたいと思っています。
(こんごは〜にとりくみたいとおもっています。)
Going forward, I'd like to work on ~.
```

```
キャリアパスについて相談させていただいてもよいですか？
(キャリアパスについてそうだんさせていただいてもよいですか？)
May I discuss my career path with you?
```

---

**Receiving feedback:**

```
承知しました。ご指摘ありがとうございます。
(しょうちしました。ごしてきありがとうございます。)

```

```

```

```
今後、〜を意識して取り組みます。
(こんご、〜をいしきしてとりくみます。)
Going forward, I'll keep ~ in mind.
```

---

**Closing:**

```
本日もありがとうございました。よろしくお願いいたします。
(ほんじつもありがとうございました。よろしくおねがいいたします。)
Thank you for today as well. I'll keep working at it.
```

## Section 4 — 30-Day Rapid Office Fluency Plan

**Goal: Functional office Japanese in 30 days. One focused daily topic.**

> **Time investment:** 20–30 minutes active study + exposure throughout the day.
> **Method:** For each day, study the assigned vocabulary, then write 3–5 sentences using those words in a realistic context.

---

### Week 1 — Foundation: Greetings + Daily Status Language

| Day | Focus | Target |
|---|---|---|
| 1 | Greeting & sign-off phrases | お疲れ様です, よろしくお願いします, 了解です, 承知しました |
| 2 | Core acknowledgment patterns | Pattern A (Section 2) — drill all 6 responses until automatic |
| 3 | Status words drill | Group 1A words + produce 5 status sentences (完了/対応中/調査中) |
| 4 | Tier S vocabulary review | All ~40 Tier S words from Section 1 — write each once, cover-recall |
| 5 | Time & deadline language | 締め切り, スケジュール, 予定, 期日, 遅延 — use in 3 sentences each |
| 6 | Standup template drill | Study 3A, write a full mock standup for your current task |
| 7 | Weekly review | Re-test Week 1 words without looking. Note what you still forget. |

**Week 1 checkpoint:** Can you say your standup without notes?

---

### Week 2 — Communication: Messages, Email & Requests

| Day | Focus | Target |
|---|---|---|
| 8 | Sharing & reporting patterns | Patterns B and D (Section 2) — drill all sentences |
| 9 | Request patterns | Pattern C — write 3 requests you commonly need to make |
| 10 | Email opener & closer | お世話になっております, よろしくお願いいたします — write a full email shell |
| 11 | Flagging problems | Pattern E — write 3 delay/problem notifications |
| 12 | Full business message | Compose a 5-sentence message: reason → detail → request → close |
| 13 | Message simulation drill | Write 5 short messages for 5 different situations without notes |
| 14 | Weekly review | Re-write all Week 2 pattern sentences from English prompts only |

**Week 2 checkpoint:** Can you write a short business message in under 3 minutes?

---

### Week 3 — Meetings & Projects: Discussion Language

| Day | Focus | Target |
|---|---|---|
| 15 | Meeting vocabulary | Group 1D — drill 15 words, then study Pattern F (clarification) |
| 16 | Progress check phrases | 進捗確認 patterns — how to give and receive progress updates |
| 17 | Project management language | Group 1F — KPI, 優先順位, 工数, 実装, リリース |
| 18 | Problem-solving language | 課題, 対策, 改善, 解決 — use each in a sentence |
| 19 | Keigo basics | Group 1H — いたす, いただく, ご〜いただく patterns |
| 20 | Meeting template | Study 3A–3B combined. Simulate a 10-minute progress meeting in writing |
| 21 | Weekly review | Compose: standup + one business message + one clarification request |

**Week 3 checkpoint:** Can you discuss a task delay with your manager in Japanese?

---

### Week 4 — Incidents & Advanced: Full Scenario Coverage

| Day | Focus | Target |
|---|---|---|
| 22 | Incident vocabulary | Group 1G — all 15 incident words, write each in a sentence |
| 23 | Incident alert templates | Study 3C Phase 1 & 2 — write a mock initial alert |
| 24 | Post-incident writing | Study 3C Phase 3 — write a resolution message + prevention plan sentence |
| 25 | 1-on-1 meeting templates | Study 3D — prepare 5 sentences for your next actual 1-on-1 |
| 26 | HR phrases | Study 3E — write a leave request and a WFH notification |
| 27 | Formal apology patterns | 申し訳ございません, ご迷惑をおかけして — write 3 apology patterns with context |
| 28 | Client & delegation phrases | Pattern G — delegation phrases + client communication vocabulary |
| 29 | Full scenario drill | Standup → problem arises → message update → resolution → 1-on-1 mention |
| 30 | Self-assessment | From each group in Section 1, pick 10 random words. Write a sentence for each. Score yourself. |

**Week 4 checkpoint (Day 30 self-test):**

Score each area out of 5:

| Area | Score |
|---|---|
| Standup — produce a full update without notes | /5 |
| Business message — write and close a request | /5 |
| Incident — write an alert and a resolution | /5 |
| 1-on-1 — discuss a blocker and a goal | /5 |
| HR — write a leave request | /5 |
| **Total** | **/25** |

- **20–25:** Ready for real business communication. Move to Phase 2 vocabulary.
- **14–19:** Strong foundation. Drill your weak scenarios for 1 more week.
- **Under 14:** Restart Week 1–2 with more daily writing practice.
